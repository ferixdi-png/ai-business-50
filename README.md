# 50 способов использовать ИИ в бизнесе

Статический сайт-презентация для предпринимателей и собственников бизнеса.

## Что внутри

- Первый экран с позиционированием продукта без инфобизнес-риторики.
- Архитектура ИИ-офиса: стратегия, финансы, маркетинг, процессы, команда/недвижимость.
- Интерактивный каталог сценариев с поиском и фильтрами.
- Готовая структура для публикации на GitHub Pages, Netlify или Vercel.

## Локальный запуск

Откройте `index.html` в браузере.

Если установлен Python:

```bash
python -m http.server 8080
```

После этого сайт будет доступен по адресу `http://localhost:8080`.

## Публикация на GitHub

На этой машине не найден GitHub CLI (`gh`). После установки и авторизации можно выполнить:

```bash
gh repo create ai-business-50 --public --source=. --remote=origin --push
```

Для GitHub Pages:

1. Откройте репозиторий на GitHub.
2. Перейдите в `Settings -> Pages`.
3. Выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
