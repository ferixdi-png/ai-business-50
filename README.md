# 50 видеоуроков по ИИ для бизнеса

Статический сайт-презентация видеопрограммы для предпринимателей и собственников бизнеса.

## Что внутри

- Черно-фиолетовый лендинг с четким первым экраном: 50 видеоуроков с прикладными бизнес-сценариями.
- Блоки “что внутри”, “кому подойдет”, “как проходить” и “формат одного урока”.
- Блок применимости: шаблон запроса, мини-задание, бизнес-результат и место внедрения.
- Простой 7-дневный маршрут для первого результата на своих данных.
- Архитектура программы: стратегия, финансы, маркетинг, процессы, команда/недвижимость.
- Интерактивный каталог уроков с поиском, фильтрами, длительностью и результатом после просмотра.
- Финальный блок с ценой `19 999 ₽` и ссылкой на Telegram `@ferixdiii`.
- Указан старт обучения `1 июля` и статус подготовки материала.
- Готовая структура для публикации на GitHub Pages, Netlify или Vercel.

## Локальный запуск

Откройте `index.html` в браузере.

Если установлен Python:

```bash
python -m http.server 8080
```

После этого сайт будет доступен по адресу `http://localhost:8080`.

## Публикация на GitHub

На этой машине не найден GitHub CLI (`gh`). После установки и авторизации можно выполнить:

```bash
gh repo create ai-business-50 --public --source=. --remote=origin --push
```

Для GitHub Pages:

1. Откройте репозиторий на GitHub.
2. Перейдите в `Settings -> Pages`.
3. Выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
